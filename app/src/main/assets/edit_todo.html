Edit or add a todo
Overview
Actions
Add todo item to project or save changes while editing a todo
Don't save changed while editing a todo or don't add a new todo when editing a new one.
Entering data
When editing a new todo or editing a existing one, the following information can be entered.
Project name:
Project to which the todo item is added. By default the current project is selected. When editing the todo item, it's also possible to move it to an other project by changing the selected project
Title
Distinguishably title of the todo item. This title is displayed in the todo overview list. The title is mandatory.
Status
Indicates status of the todo item such as : Not active, bussy with todo , finished or for example removed
Information how the list of available statuses can be configured: Configuring status selection
'Should start on ' and 'Should finish on '
The dates when you should start or finish this todo. In the todo overview list the "start date" turn green on or after the start date
After the finish date, the "finish date" turns green
Comment
Any information about the todo item.
