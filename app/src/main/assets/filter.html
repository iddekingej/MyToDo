Todo filters
See also
Todo overview
Function:
A list of user defined Todo filters.
In the todo overview list a filter can be selected so only relevant todo's are displayed. Those filters are user defined.
Operation:
Those filters can be defined here as follows:
  • :Add a new filter: By pressing the this icon a form is open for editing a new filter. For more information see Add or edit filters
  • Edit a filter: By tapping the filter a form opens for editing a filter . For more information see Add or edit filters
  • Delete a filter: Goto edit filter by clicking the filter. On the top of the screen there is a delete icon:
