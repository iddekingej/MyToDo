Todo filter
In the todo overview list a filter can be selected so only relevant todo's are displayed. Those filters are user defined.

At "todo filters" all defined filters are displayed
The following action can be done here:
  • Add a new filter by pressing the ... icon. For more information see Add or edit filters
  • Edit a filter by clicking the filter. For more information see Add or edit filters

Filters can be deleted when editing a filter, for more information see Add or edit filters
