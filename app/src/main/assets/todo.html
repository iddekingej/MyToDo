At this screen a list with todo's belogning to a project are displayed. On top of the screen there is a selection list for filters. Filters are user defined, see for more information :filters.
At this screen the following operation are possible:
Add a new todo
Showing todo details Tap the todo,details of the todo item is displayed
Edit todo: tap the todo item, a screen is displayed with todo details.
A new todo can be added by pressing the '+' button on the top of the screen.
By tapping the todo, a screen with details about the todo is openend (from this screen it's possible to edit the the todo by pressing the ... icon).
