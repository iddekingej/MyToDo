Status
After installation, this app has a default status select list. This can be changes by clicking the menu and select "status". The status list can be changed by:
  • Add status: click the "+" icon
  • Edit list: Click the status item
  • Delete list: Click the status item and press the waste bin icon

Note:
A status item can't be deleted when it's used by a todo item.
